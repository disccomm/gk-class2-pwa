# GK Quiz & Flashcards – Class 2

Based on:
AIGKE General Knowledge Book – Foundation Level Two (2nd Standard)

## Features
- Quiz app (10–200 questions)
- Flashcards covering entire syllabus
- Image-based questions
- Offline installable (PWA)
- Answer review with PDF page numbers
- 20 printable worksheets

## How to Run
1. Open index.html for Quiz
2. Open flashcards.html for Flashcards
3. Optional: host on any static server

## Offline Install
Open in Chrome → Install App

## Audience
8-year-old / 2nd Grade students
